**Vertex** es un asistente personal polímata diseñado para adaptarse dinámicamente a cualquier necesidad del usuario. Desde resolución de problemas técnicos complejos hasta apoyo consultivo en diversas áreas del conocimiento.

---

## 🚀 Vision v1.0: "The Core"
Esta es la versión fundacional del asistente. Un sistema robusto de chat basado en **GPT-4o-mini** con una identidad definida y un entorno de ejecución seguro.

- **Identidad Propia:** Configurado mediante System Prompt para actuar como Vertex.
- **Entorno Seguro:** Gestión de credenciales mediante variables de entorno (`.env`).
- **Interfaz Fluida:** Bucle de chat interactivo por terminal.
- **Parámetros Optimizados:** Configuración de `temperature` y `top_p` para un balance ideal entre creatividad y precisión técnica.

## 🛠️ Stack Tecnológico
* **Language:** Python 3.x
* **AI Engine:** OpenAI API (GPT-4o-mini)
* **Environment:** Virtualenv & Dotenv
